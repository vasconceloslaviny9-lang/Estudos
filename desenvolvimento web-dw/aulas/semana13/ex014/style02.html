CSS interno

Um estilo interno é aquuele que fica configurado dentro da área do <Head> do seu documento.

A grande vantagem dessa maneira em relação o modelo inline é que podemos escrever uma configuração única para todos os elçementos da mesma família. Por exemplo, todos os paragrafos deste documento esão com a mesma configuração de cor, e eu só preciso escrever isso uma vez.

A grande desvantagem deste modelo em relação a ultima estrutura que veremos é que no CSS externo (a estrutura do próximo exercício) é que o estilo não pode ser reaproveitado por outras páginas.

IMPORTANTE: Você pode misturar CSS inline com CSS interno em um mesmo documento sem problemas! Vamos fazer um pequeno TESTE! É deste processo de aplicamos os estilos em diferentes níveis que surge o conceito de "Folhas de Estilo em Cascata".
