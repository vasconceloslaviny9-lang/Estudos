No livro PHP: a Bíblia, Tim Converse explica o que é PHP.

O PHP é uma linguagem para a criação de scripts para a Web do lado servidor embutidos em HTML, cujo código-font é aberto, e que é compatível com os mais importantes servidores Web (especialmente Apache). O PHP permite incorporar fragmentos de código em páginas HTML normais - códigos esse que é interpretado à medida que suas páginas são oferecidas aos usuários. O PHP também serve como um linguagem que "cola", facilitando a conexão de suas páginas Web com o branco de dados do lado servidor.
